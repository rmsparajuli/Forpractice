Data Types

    Primitive Types and Object Types

  1. Primitive Types

    1. Defination

      • immutable data types (which cannot be change)
      • for eg: let x=2, x=3; x value stays 3

    2. String

      • String is just a siple text data.
      • We can specify string in the single or double quotes or you can use both.
      • eg: let text = "This is your string"

    3. Boolean

      • Boolean is just a value type that have only two possible values.(true or false)
      • don't specify the Boolean type in the Quotation mark like this. (" ")

    4. Symbols

      • In ecmascript 6 we have a new type symbol.
      • Symbols is the new data type in es6 which represent Unique tokens.
      • Once you create a symbol it is unique and it will not match any other symbols

    5. Null and Undefined

      • JavaScript has two special types null and undefined.
      • Now has only one possible value which is Null and undefined has only one possible value defined.
      • Both null and undefined Represent something that doesn't exist.
      • ie: null > no data is defined
      • ie: undefined > only variable is defined without value

  2. Object Types

    1. Defination

      • Objects are mutable.
      • Can change any value anytime.
      • Object is just like a Container that can have different king of data.
      • Object literal syntax : {}
      • example: const obj1 = {}

    2. Array

      • Array is Collection of Different types of data.
      • Array Start with [0] and end with [n] value.
      • Important Features of Array
        1. Array element can be any type of data. it may be objects or primitive type.
        2. Array size is not fix so you can add and remove array element anytime.
        3. Array is zero based. so the first element of the array is zero the next element is 1 and so on.

    3. Date

      • So dates are represent the date and time using date object.
      • Date object used to get date or you can use to set date for a particular event.
      • you can also specify time using date of object

    4. Regular Expression

      • Regular expression is sub language of JavaScript.
      • it is used to Perform complex search and replace operations on strings
      • Once you create a symbol it is unique and it will not match any other symbols.

    5. Map and sets

      • In new ecmascript 6 a new data type introduced is map and set And their week counterpart, WeakMap and WeakSet.
      • Mpas are just like an object but it Offers more advantages then object in certain situations.
      • Set are similar to array except they can't content dublicate element.
