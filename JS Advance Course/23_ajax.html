Ajax

  1. Defination

    • Asynchronous JavaScript and XML.
    • it describes a set of developement techniques used for building websites and web application
    • Basically the use of ajax is to update web content Asynchronously.
    • Asynchronously: which means user's web browser donot need to reload entire browser to make change

  2. "J" in AJAX stands for JavaScript

    • it's used specifically to create, add, and manage dynamic website content.
    • JavaScript is then used to control features that require real time updates while a visitor is viewing a page

  3. "X" in AJAX is XML (Extensible Markup Language)

    • While HTML and CSS focus on instructing how page content is displayed, XML is used to transfer data stored on the page to the browsers that view it.
    • Ajax uses XHTML for content, CSS for presentation, along with Document Object Model and JavaScript for dynamic content display.
    • A user can continue to use the application while the client program requests information from the server in the background.
