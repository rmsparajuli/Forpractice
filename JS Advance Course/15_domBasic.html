DOM

  1. About DOM

    • Dom stands for document object Model.
    • According to the document object model all tag in HTML is an object.
    • The text specified in the tag is also an object.
    • So all these objects are accessible using javascript.
    • DOM is a standard object model and programming interface for HTML.
    • The DOM object defines the HTML elements as object
    • Properties of HTML elements.
    • Methods to access HTML elements and Events for HTML element.

  2. DOM Tree

    • Conceptually, Dom is a tree.
    • A tree consists of nodes.
    • Every notes has parents, zero or more child notes.
    • The root node is document, has only one child HTML.
    • The HTML element has two children title and head

  3. What you can do using DOM Object

    • You can change any HTML elements and its attributes.
    • You can create new HTML elements with its attributes and you can also remove it if you want
    • You can add and change CSS styling of the page.
    • Can react all the event in the HTML page and you can create or change or change any evemt in the webpage.

  4. DOM Programming Interface

    • All HTML elements are defined as objects.
    • In the programming interface property is a value which you can get or set.
    • Methods are action perform on the HTML element (deleting a HTML element or create a new one).
