OOP CONCEPT

  1. Object

    • Object is a simply a unoudered collection of properties. Each of which has a name and value.
    • Object can aggregate multiple values and allows you to store and access that value by property name
    • Property name can be a string or symbol

  2. Object Oriented Programming

    • In object oriented programming object is a logically related Collection of data and functionality
    • Now this object can be a collection of properties and methods.
    • Lets take Example: Object ={Car}, Properties ={ all parts of car } Methods = {man driving a car}
    • To create object we should have Properties and Methods

  3. Obect Reference

    • When you specify your object reference to the other variable then you can modify this object using the reference variable.
